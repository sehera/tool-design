This tool aims to create an experience with design that isn't about the end product
but the interactiveness and abstract form. I wanted to make a calm yet abstract space
where the user would input clicks and create a randomized geometric pattern. I think
the tool is quite calming in some ways.

The tool offers other modes; new colors, a square mode and a random color mode
which are all collected and calm in their design. The randomized mode offers
more 'fun' colors but keeps intact the simplicity of the experience.
